## 快速幂算法

### 基本原理

计算x的n次方， 即计算$$x^n$$。

由公式可知： $$x^n = x^{n/2} * x^{n/2}$$。

如果我们求得 $$x^{n/2}$$， 则可以O\(1\)求出x^n, 而不需要再去循环剩下的n/2次。

以此类推，若求得 $$x^{n/4}$$， 则可以O\(1\)求出 $$x^{n/2}$$。  
。。。

因此一个原本O\(n\)的问题，我们可以用O\(logn\)复杂度的算法来解决。

### 递归版本的快速幂算法

Python:

```py
def power(x, n):
    if n == 0:
        return 1
    
    if n % 2 == 0:
        tmp = power(x, n // 2)
        return tmp * tmp
    else:
        tmp = power(x, n // 2)
        return tmp * tmp * x
```

注意:

* 不要重复计算，在计算 $$x^{n/2} * x^{n/2}$$的时候，先计算出 $$x^{n/2}$$，存下来然后返回 $$tmp*tmp$$;
* n为奇数的时候要记得再乘上一个x。
